    <p>Welcome to the GitHub repository for your friendly virtual assistant! This program retrieves the date, current time, offers a greeting, and allows you to search for information about a person.</p>

    <h2>Features</h2>
    <ul>
        <li><strong>Date and Time:** Displays the current date and time.</li>
        <li><em>Personalized Greetings:** Generates a greeting based on the time of day.</li>
        <li><u>Person Search:** Allows you to search for information about a person (implementation details will depend on your chosen approach).</li>
    </ul>

        </li>
        <li>**Install Dependencies:**

    <h2>Usage</h2>
    <hr>
